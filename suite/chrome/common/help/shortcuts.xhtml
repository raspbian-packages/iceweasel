Använda kortkommandon

Detta dokument använder följande format för att lista kortkommandon:

Exempel:

Kommando Windows Mac OS Linux eller Unix
Kopiera Ctrl+C Cmd+C Ctrl+C

För att utföra ett kommando, tryck på de tangenter som listas tillsammans samtidigt. T.ex., för att kopiera i Windows, tryck samtidigt på Ctrl och C.

Tips: Plustecknet (+) betyder att du måste trycka på de två tangenter som omger plustecknet samtidigt. Om det finns fler än ett plustecken (t.ex. Ctrl+Skift+C), betyder det att alla tre tangenter som omger plustecknen måste tryckas ner samtidigt.

Förkortningar:

  • Ctrl = Control-tangenten
  • Cmd = Kommando-tangenten på Mac OS tangentbord

OBS! Vissa kortkommandon utför olika funktioner beroende på var musmarkören befinner sig (fokus). T.ex., om du trycker Home i Windows när du visar en webbsida kommer &brandShortName; att flytta till början på webbsidan. Men om du trycker Home i Windows när markören befinner sig i ett textfält kommer istället markören att flyttas till början av textfältet.
