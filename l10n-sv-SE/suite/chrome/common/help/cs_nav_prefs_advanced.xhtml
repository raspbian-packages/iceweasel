  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Avancerat, klicka på Skript/Insticksmoduler. (Om inga underkategorier är synliga, dubbelklicka på Avancerat för att veckla upp listan.)

Inställningspanelen Skript & Insticksmoduler låter dig styra hur JavaScript och insticksmoduler används:

  • Aktivera JavaScript för: Markera dessa alternativ för att aktivera JavaScript i:
    • Webbläsare: Växlar JavaScript av och på i webbsidor öppnade i webbläsaren.
    • E-post & Grupper: Växlar JavaScript av och på i webbsidor öppnade i E-post & Grupper.
  • Tillåt skript att: Markera dessa alternativ för att styra hur JavaScript får användas:
    • Flytta eller ändra storlek på fönster: Tillåt att öppna fönster kan flyttas eller få sin storlek ändrad.
    • Höja eller sänka fönster: Tillåt att fönster kan placeras under eller över andra fönster.
    • Dölja statusfältet: Tillåt att statusfältet kan döljas.
    • Byta text i statusfältet: Tillåt att text i statusfältet kan ändras, såsom med rullande text i statusfältet.
    • Byta ut bilder: Tillåt att bilder kan bytas ut eller animeras, såsom med bildväxling (bilder som byts ut när du placerar muspekaren över dem).
    • Blockera eller ersätta snabbvalsmenyer: Tillåt att högerklicksmenyer eller, om du använder en mus med en knapp, Ctrl-klickmenyer kan ersättas eller blockeras av webbsidor.
  • Aktivera insticksmoduler för: Markera detta alternativ för att styra hur insticksmoduler får användas:
    • E-post & Grupper: Tillåter insticksmoduler att användas i E-post & Grupper.

För mer information om insticksmoduler, se Insticksmoduler och filhämtningar.

Inställningspanelen Cache låter dig justera &brandShortName;s minnes- och diskcache:

    • När sidan har blivit för gammal: Pricka för detta om du vill att &brandShortName; ska jämföra en webbsida med vad som finns sparat i cachen, när sidan enligt servern har förfallit.

    • Aldrig: Pricka för detta om du inte vill att &brandShortName; ska jämföra cachad information med nätverket.
  • Förhandshämta webbsidor vid inaktivitet, så att länkade sidor i för förhandshämtning anpassade webbsidor kan läsas in snabbare: Markera detta för att minska tiden det tar att visa en webbsida när du klickar på en länk på en webbsida som använder förhandshämtning. För mer information om denna funktion, se det webbaserade Link Prefetching FAQ.

Inställningspanelen Proxyservrar låter dig ställa in &brandShortName; att använda sig av en proxyserver:

Innan du börjar: Fråga din nätverksadministratör om det finns en proxykonfigurationsfil eller vilket namn och portnummer du ska använda.

  • Direktansluten till Internet: Välj detta om du inte vill använda någon proxyserver.
  • Automatisk identifiering av proxyinställningarna: Välj detta om du vill låta &brandShortName; automatiskt leta efter och konfigurera proxyinställningarna med hjälp av WPAD-protokollet.
  • Automatisk proxykonfiguration, URL: Välj detta om du har en proxy-konfigurationsfil eller en URL. Skriv sedan in konfigurations-URL:en.
    • Uppdatera: Klicka på denna för att uppdatera konfigurationsfilen.

      För mer information om automatisk proxykonfiguration, se den webbaserade användarguiden End User Guide to Proxy AutoConfiguration.

  • Manuell proxykonfiguration: Välj detta om du inte har en proxyadress (URL) eller om den automatiska proxyidentifieringen inte kunde konfigurera proxyn riktigt.
    • Proxy: Skriv in namnet eller den numeriska IP-adressen till proxyservern.
    • Port: Skriv in portnumret i fältet Port. Klicka på Avancerat för att ställa in Avancerade proxyinställninga
    • Undantag: Skriv in domäner och/eller IP-adresser för vilka du inte vill använda någon proxyserver. Avskilj varje post med ett komma. (Exempel: .mozilla.org, .net.nz, 192.168.1.0/24.)

Avancerade proxyinställningar

Om du vill använda olika proxyservrar för olika protokoll eller behöver använda en SOCKS-proxy:

  • HTTP-proxy, SSL-proxy, FTP-proxy, Gopher-proxy: Skriv in namnet eller den numeriska IP-adressen till proxyservern. Skriv in porten i portfälten.

    OBS! Om du använder samma inställningar för alla typer av proxyservrar, klicka på Använd inställningarna för HTTP-proxyn för alla protokoll.

  • SOCKS-proxy: Skriv in namnet eller den numeriska IP-adressen till proxyservern. Skriv in porten i portfältet.
    • SOCKS v4, SOCKS v5: När du skriver in en SOCKS-proxy, pricka för SOCKS v4 eller SOCKS v5 beroende på vilken version av SOCKS som används av proxyservern.
    • Använd för att slå upp värdnamn: Markera detta för att använda SOCKS-proxyn för att slå upp värdnamn. Detta rekommenderas för SOCKS v5-proxyservrar.

Avancerade inställningar - HTTP-anslutning

Detta avsnitt beskriver hur du använder inställningspanelen HTTP-anslutning. Om du inte redan ser den panelen, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Avancerat, klicka på HTTP-anslutning. (Om inga underkategorier är synliga, dubbelklicka på Avancerat för att veckla upp listan.)

Inställningspanelen HTTP-anslutning används för att konfigurera HTTP-baserad nätverkskommunikation:

  • Direktanslutningsalternativ, Proxyserveranslutningsalternativ: Välj HTTP-version och alternativ för direkta anslutningar och proxyserveranslutningar.
    • Använd HTTP 1.0: Välj detta för att använda originalversionen av HTTP, standardiserad 1996.
    • Använd HTTP 1.1: Välj detta för att använda den nya versionen av HTTP, vilken erbjuder förbättringar av prestanda, såsom effektivare användning av HTTP-anslutningar, bättre stöd för klientbaserad caching, samtidiga HTTP-förfrågningar (pipelining) och större kontroll över cache som tar slut och ersättningsprinciper.
    • Aktivera Keep-Alive: Markera detta för att hålla en anslutning öppen för ytterligare HTTP-förfrågningar, vilket ökar snabbheten.
    • Aktivera Pipelining: Markera detta för att aktivera pipelining, vilket möjliggör för mer än en HTTP-förfrågan att skickas till servern åt gången, vilket minskar fördröjningar vid inläsning av webbsidor.

      OBS! Pipelining är endast möjlig med HTTP 1.1.

Avancerade inställningar - Mushjul

Detta avsnitt beskriver hur du använder inställningspanelen Mushjul. Om du inte redan ser den panelen, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Avancerat, klicka på Mushjul. (Om inga underkategorier är synliga, dubbelklicka på Avancerat för att veckla upp listan.)

Inställningspanelen Mushjul låter dig ange hur mushjulet (mellan musknapparna) på din mus ska användas i &brandShortName;. Moderna möss kan ha två hjul eller en knapp som kan användas för att byta rullriktning för hjulet. Funktionen för det vertikala hjulet anges i den övre panelen Vertikal rullning medan det horisontella läget styrs av den nedre panelen Horisontell rullning.

  • Välj Ingen ändringstangent eller välj en tangent som du vill använda tillsammans med mushjulet. Använd alternativknapparna nedan för att konfigurera mushjulsfunktioner:
    • Rulla dokumentet med: Välj detta om du vill rulla dokumentet med det antal rader (eller tecken för horisontell rörelse) som du angett i fältet.
      • Använd systemstandard: Välj detta om du vill använda systemets förvalda inställning — de antal rader (tecken) du kanske tidigare har angett kommer att åsidosättas.
    • Rulla en sida upp (åt vänster) eller en sida ned (åt höger): Välj detta om du vill rulla upp eller ned en sida i taget. Denna inställning erbjuder snabbare men mindre exakt rullning igenom en sida med mushjulet.
    • Gå bakåt eller framåt eller i historiken: Välj detta om du vill använda mushjulet för att navigera bakåt eller framåt till tidigare besökta sidor.
    • Gör texten större eller mindre: Välj detta om du vill använda mushjulet för att öka eller minska storleken på texten på en webbsida. Denna inställning kan hjälpa dig att lättare läsa en sida eller göra så att texten bättre passar in på skärmen.

    OBS! Varje ändringstangent kan ges en egen funktion.

  • Om din mus inte har ett läge för horisontell rullning kommer alla inställningar i den nedre panelen Horisontell rullning att ignoreras.

Avancerade inställningar - DOM Inspector

DOM Inspector är en valfri webbutvecklingskomponent. Det här avsnittet beskriver hur du använder dess inställningspanel. Om du inte redan ser den panelen, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Avancerat, klicka på DOM Inspector. (Om inga underkategorier är synliga, dubbelklicka på Avancerat för att veckla upp listan.)

När du väljer ett element, kan DOM Inspector automatiskt markera det genom att blinka detsamma eller dess ram:

  • Blinka det valda elementet: Markera detta alternativ för att aktivera markeringen av ett valt element. Som förval kommer en blinkande ram att dyka upp kring elementet.
    • Ramfärg: Välj en färg på ramen runt elementet.
    • Rambredd: Ange bredden på ramen runt elementet.
    • Blinktid: Ange tiden (i millisekunder) som du vill att blinkandet ska pågå.
    • Blinkhastighet: Ange tidsintervallet (i millisekunder) mellan blinkningarna.
    • Invertera färger: Markera detta alternativ för att belysa det valda elementet med den inverterade ramfärgen. Detta får hela elementet — inklusive dess ram — att blinka.
