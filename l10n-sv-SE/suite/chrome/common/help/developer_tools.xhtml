Verktyg för webbutvecklare

&brandShortName; erbjuder flera verktyg som kan hjälpa webbutvecklare att skapa och avlusa webbapplikationer. Några av dessa väljer du själv om du vill installera. För att få åtkomst till verktygen, öppna verktygsmenyn och välj Webbutveckling.

  • Felkonsol: ett konsolfönster i vilket rapporteras problem med JavaScript och CSS-kod i webbapplikationer samt i &brandShortName;-programmet självt. Som förval redovisas CSS-körningsfel och JavaScript-fel. Konsolen kan även användas för att visa loggade meddelanden om XUL- och JavaScript-kod.
  • DOM Inspector: Ett verktyg som kan användas för att granska och redigera objektmodellen (DOM) i ett webbdokument eller en XUL-applikation.

    Webbaserade resurser:

    • Introduction to the DOM Inspector (Ian Oeschger)
    • DOM Inspector FAQ (mozilla.org)
    • Creating Applications with Mozilla - Appendix B3: The DOM Inspector
  • Avlusare för JavaScript: Även känd som Venkman. Med detta verktyg kan du avlusa JavaScript i &brandShortName;. Det stöder in- och urstigning, avbrottspunkter och många andra funktioner.

    Webbaserade resurser:

    • Venkman JavaScript Debugger project page
    • Learning the JavaScript debugger Venkman (Svend Tofte)
    • Debugging JavaScript Using Venkman (webreference.com, artikelserie)
    • Venkman Development Page
