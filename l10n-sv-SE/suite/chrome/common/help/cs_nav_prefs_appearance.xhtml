Inställningar för Utseende - Utseende

Detta avsnitt beskriver hur du använder inställningspanelerna för Utseende. Om du inte redan ser en av dessa paneler, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Klicka på kategorin Utseende.

Inställningspanelen Utseende låter dig ange startalternativ för &brandShortName; och anpassa gränssnittet:

  • När &brandShortName; startar, öppna: Markera de komponenter du vill använda när du startar &brandShortName;
  • Visa verktygsfält som:
    • Bilder och text: Pricka för detta för att se text under varje knapp i verktygsfältet.
    • Endast bilder: Pricka för detta för att endast visa knapparna i verktygsfältet.
    • Endast text: Pricka för detta för att endast visa textknappar.
  • Visa verktygstips: Markera detta om du vill att verktygstips ska dyka upp när muspekaren placeras över delar av &brandShortName;s eller på vissa webbsidor.
  • Gränssnittsspråk: Denna inställning låter dig byta ut språket som används i &brandShortName;s användargränssnitt. Ytterligare språk kan installeras från &brandShortName;s hemsida. OBS! Du måste starta om &brandShortName; för att en ny språkinställning ska få effekt.

Inställningar för Utseende - Innehåll

Detta avsnitt beskriver hur du använder inställningspanelen för Innehåll. Om du inte redan ser den panelen, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Utseende, klicka på Innehåll. (Om inga underkategorier är synliga, dubbelklicka på Utseende för att veckla upp listan.)

Inställningspanelen Innehåll låter dig ändra inställningar som påverkar hur webbplats- och meddelandeinnehåll visas i &brandShortName;.

  • Webbplatsikoner:
    • Visa webbplatsikoner: Markera detta om du vill se webbplatsspecifika ikoner, ifall de är tillgängliga, i stället för bokmärkesikonen . Webbplatsikoner visas till vänster om både adressfältet och webbläsarens flikar.
    • Leta aktivt efter webbplatsikoner när sidan inte anger någon: Om sidan självt inte anger en webbplatsikon, innebär aktiverandet av denna inställning att &brandShortName; letar efter en "adressikon" på servern och använder den i stället.
    • Visa webbplatsikoner i bokmärkesmenyn och verktygsfältet:
      • Visa aldrig ikoner för bokmärken: Markera detta om du bara vill se de förvalda ikonerna och inte webbplatsens egen ikon i bokmärkesmenyn eller det egna verktygsfältet.
      • Bara om webbsidan nyligen var inläst: Markera detta för att visa webbplatsens egen ikon bara om webbsidan nyligen varit inläst, och ikonen för närvarande återfinns i webbläsarens cache.
      • Hämta alltid webbplatsikoner för bokmärken: Markera för detta för att alltid hämta webbplatsikoner för att visas i bokmärkesmenyn eller det egna verktygsfältet, även om de inte finns i cachen.
  • Använd mjuk rullning: Markera detta för att aktivera mjuk rullning. Om du trycker på Page Down-tangenten när detta är aktiverat kommer sidan — i stället för att göra ett omedelbart hopp — att mjukt rullas ned till nästa sida.
  • Förminska stora bilder så att de ryms i webbfönstret: Markera för detta om du vill att &brandShortName; automatiskt ska krympa stora enstaka bilder så att de ryms i webbfönstret. Att klicka på den förminskade bilden gör att den återfår sin ursprungliga storlek.
  • Zooma bara text i stället för hela sidor: Markera detta om du vill att &brandShortName; bara ska ändra storleken på texten på webbsidor när "Zooma"-funktionen används. Om detta inte är markerat kommer hela sidan inklusive bilder att zoomas.

OBS! Se de separata inställningspanelerna för färg, teckensnitt och språk för att ytterligare anpassa utseendet på innehållet, och sektionen Integritet och säkerhet för integritetsrelaterade inställningar, vilka också kan påverka hur innehåll framträder för dig.

Inställningar för Utseende - Teckensnitt

Detta avsnitt beskriver hur du använder inställningspanelen Teckensnitt. Om du inte redan ser den panelen, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Utseende, klicka på Teckensnitt. (Om inga underkategorier är synliga, dubbelklicka på Utseende för att veckla upp listan.)

Inställningspanelen Teckensnitt låter dig ange sidors teckensnitt och format.

OBS! Vissa teckenformat kan kanske inte användas därför att det valda språket inte har teckensnitt tillgängliga för det formatet.

  • Teckensnitt för: Välj en språkgrupp/skrift. Om du t.ex. vill ange förinställda teckensnitt för västeuropeiska språk/skrift (latin), välj Västerländska språk. För ett språk/skrift som inte finns i listan, välj Andra språk.
    • Variabelt breddsteg: Välj om text med variabelt breddsteg ska vara antikva (som Times Roman) eller sanserif (som Arial). Du kan även ange vilken teckensnittsstorlek du vill använda för variabelt breddsteg. Texten varierar i bredd, så tecken och bokstäver är olika breda.
    • Antikva: Välj ett teckensnitt av antikvatyp som du vill använda för webbsidor.
    • Sanserif: Välj ett teckensnitt av sanseriftyp som du vill använda för webbsidor.
    • Skript: Välj ett teckensnitt av handskriftsliknande typ som du vill använda för webbsidor.
    • Extrem: Välj ett teckensnitt av fantasityp som du vill använda för webbsidor.
    • Fast breddsteg: Välj ett teckensnitt med fast breddsteg (som Courier) som du vill använda för webbsidor. Text med fast breddsteg är fixerad i bredd, så varje tecken eller bokstav tar upp lika stor bredd.
    • Minsta teckensnittstorlek: Välj den minsta teckensnittstorlek som du vill ha visad på webbsidor.
  • Tillåt dokument att använda andra teckensnitt: Markera denna ruta om du vill behålla en webbsidas inställningar för teckensnitt och format i stället för dina egna inställningar.

Inställningar för Utseende - Färger

Detta avsnitt beskriver hur du använder inställningspanelen Färger. Om du inte redan ser den panelen, följ dessa steg:

  1. Öppna menyn &brandShortName; Redigera och välj Inställningar.
  2. Under kategorin Utseende, klicka på Färger. (Om inga underkategorier är synliga, dubbelklicka på Utseende för att veckla upp listan.)

Inställningspanelen Färger låter dig ange färger för bakgrund och text på webbsidor:

  • Text och bakgrund: Klicka på de färgade blocken för att välja en färg för visning av text och bakgrund på webbsidor.
    • Använd systemfärger: Markera detta om du vill använda dina systemfärginställningar.
  • Länkfärger: Klicka på de färgade blocken för att välja en färg för visning av obesökta, aktiva och besökta länkar på webbsidor.
    • Stryk under länkar: Markera detta för att visa understrukna länkar på webbsidor.
  • Om en webbsida har egna färger och egen bakgrund:
    • Använd alltid färger och bakgrund som anges av webbsidan: Tillåter webbsidan att välja egna färger och bakgrunder.
    • Använd de färger jag valt och ignorera de angivna färgerna: Låter dig välja färger, ignorerar webbsidans färger och bakgrundsbild.
